1. CREACIÓN Y ACTIVACIÓN DEL ENTORNO VIRTUAL
--------------------------------------------------

venv\Scripts\activate


--------------------------------------------------
2. INSTALACIÓN DE DEPENDENCIAS
--------------------------------------------------

Se instalaron las dependencias necesarias:

pip install pytest
pip install coverage
pip install -r requirements.txt

--------------------------------------------------
3. CREACIÓN DE CARPETA DE PRUEBAS
--------------------------------------------------

Se creó la carpeta de pruebas en la raíz del proyecto:

mkdir test

--------------------------------------------------
6. EJECUCIÓN DE PRUEBAS UNITARIAS
--------------------------------------------------

Se ejecutaron todas las pruebas unitarias:

python -m pytest app/test -v

Resultado esperado:

Todas las pruebas en estado PASSED


--------------------------------------------------
7. ANÁLISIS DE COBERTURA
--------------------------------------------------

Se limpió la información de cobertura anterior:

python -m coverage erase

Se ejecutaron las pruebas con análisis de cobertura:

python -m coverage run -m pytest app/test -v

Se generó el reporte de cobertura en consola:

python -m coverage report -m


--------------------------------------------------
8. GENERACIÓN DE REPORTE HTML DE COVERAGE
--------------------------------------------------
Se generó el reporte visual en HTML:

python -m coverage html

Se abrió el reporte en el navegador (Windows):

start htmlcov/index.html
